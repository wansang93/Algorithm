```python
from collections import deque

N = int(input())
q = deque()
for i in range(1, N+1):
    q.append(i)

while len(q) != 1:
    q.popleft()
    q.append(q.popleft())

print(q.pop())
```

해당 문제의 특징을 이용한 풀이

길이가 홀수면 가장 뒤에 + 1번부터 2칸씩

길이가 짝수면 1번부터 2칸씩

```python
N = int(input())
lst = [i for i in range(1, N+1)]

while len(lst) > 1:
    if len(lst) % 2:
        lst = [lst[-1]] + lst[1::2]
    else:
        lst = lst[1::2]

print(*lst)
```
