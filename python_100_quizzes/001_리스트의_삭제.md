# [문제1 : 리스트의 삭제](https://www.notion.so/af306f9659184d599c404b2dcb86a480)

다음 리스트에서 400, 500를 삭제하는 code를 입력하세요.

```python
nums = [100, 200, 300, 400, 500]
```



# 풀이1-1

리스트의 마지막 값을 2번 삭제한다.

``` python
nums.pop()
nums.pop()
```



# 풀이1-2

리스트의 3, 4번째 값을 삭제한다.

``` python
del nums[4]
del nums[3]
```



# 풀이1-3

리스트의 값 400, 500을 삭제한다.

``` python
nums.remove(400)
nums.remove(500)
```

- 값이 2개 이상인 경우 가장 앞에있는 값 한개를 삭제한다.
- 값이 없는 경우 오류가 발생한다.
