# [문제16 : 로꾸거](https://www.notion.so/16-29e7bb0f4fea40b0ac28713eadd2be3a)
문장이 입력되면 거꾸로 출력하는 프로그램을 만들어 봅시다.

# 풀이16-1
``` python
sentence = input()
print(sentence[::-1])
```

# 풀이16-2
``` python
s = input()
r_s = ""
for i in range(len(s)):
    r_s += s[len(s)-i-1]

print(r_s)
```

# 풀이16-3
``` python
sentence = input()
print(''.join(list(reversed(sentence))))
```


- 리스트 인덱싱
사용법은 `리스트변수[시작인덱스:종료인덱스:step]` 입니다.
