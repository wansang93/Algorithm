# [문제61 : 문자열 압축하기](https://www.notion.so/61-48803caf1f2141afa859b98b058f50e3)

문자열을 입력받고 연속되는 문자열을 압축해서 표현하고싶습니다.

**입력**

    aaabbbbcdddd

**출력**

    a3b4c1d4


# 풀이61-1

``` python
mystr = input()
newstr = mystr[0]

count = 0
for c in mystr:
    if newstr[-1] == c:
        count += 1
    else:
        newstr += str(count)+c
        count = 1

newstr += str(count)

print(newstr)
```

# 풀이61-2

``` python
import re

input_data = input()
three = re.findall('(\\w)(\\1*)', input_data)

s = ''
for i, j in three:
    s += str(len(j)+1)+i
print(s)
```
