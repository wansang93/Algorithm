# [문제15 : 자기소개](https://www.notion.so/15-c9945a98393f48a081ef43be544e884f)

신학기가 시작되고, 아이들이 돌아가면서 자기소개를 하기로 했습니다.
만약 입력으로 `김다정`이라는 이름이 주어지면
    안녕하세요. 저는 김다정입니다.
라고 출력하게 해주세요.

# 풀이15-1
``` python
name = input()
print("안녕하세요. 저는 " + name + "입니다.")
```

# 풀이15-2
``` python
name = input()
print("안녕하세요. 저는 %s입니다." % name)
```

# 풀이15-3
``` python
name = input()
print("안녕하세요.", "저는 %s입니다." % name)
```
